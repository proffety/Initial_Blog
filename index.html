O que é o HTML

O HTML é uma linguagem de marcação criada para ser o "esqueleto" de uma página web criado por Tim Berners-Lee, o conceito de uma página html envolve a estrutura básica com tags, sendo a estrutura básica algo assim: 

            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Document</title>
            </head>
            <body>
                
            </body>
                </html>

Introdução as tags HTML

As tags html são divididas em 2 tipos e tem a seguinte estrutura: elas começam com o sinal de menor que <, em seguida temos o nome da tag como a tag p, após a tag deve ser finalizada com o sinal > formando algo como <p>, os tipos de tags HTML são as que fecham em si, e as que fecham em tags, exemplo de uma que fecha em tag é a propria tag p, para fechar uma tag se faz </p> o sinal de / diz que uma tag fechou e tudo entre a abertura é considerado conteudo
exemplo: <p> aqui é o conteudo </p>
o outro tipo de tag são as tags que fecham nela mesma, isto é a tag não tem conteudo, pois ela é o proprio conteudo exemplo <img src = " " /> a tag img abre e fecha sozinha, ela não tem uma dupla, pois ela é o proprio conteudo, ela é a imagem em si, e não precisa de um conteudo que seja a imagem (pois ela ja é)

Tags básicas de estruturação

as tags básicas para a estruturação de um site são as tags h que vai do 1 ao 6 e são tags de titulos, p que são tags de paragragos, e img que são para imagens
a tag h tem uma particularidade: ela tem uma numeração que varia <h1> até <h6>, onde a numeração indica a hierarquia do conteudo sendo h1 a hierarquia maxima (titulo principal ou principais) e h6 a hierarquia minima (titulo menos relevante)
ps - não se deve pensar como erroneamente é pensado sobre as tag's h, que é pensar como eles tendo funções de aumentar ou diminuir titulos, a estilização é responsabilidade das css, as tags h tem função de hierarquia semantica
